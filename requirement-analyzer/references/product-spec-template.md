## 5. 功能清单【必填，按页面拆分描述】

**格式说明**：每个功能下的每个页面单独描述，不糅杂到一起

### 功能1：XXX功能（高优先级）

#### 1. 列表页面

##### 1.1 页面按钮
- 新增按钮
- 编辑按钮
- 删除按钮
- 导出按钮

##### 1.2 页面查询条件
| 字段名称 | 控件类型 | 说明 |
|----------|----------|------|
| 场所名称 | 文本框 | 模糊查询 |
| 场所分类 | 下拉框 | 选项：企业/个体/政府 |

##### 1.3 页面展示字段
| 字段名称 | 控件类型 | 说明 |
|----------|----------|------|
| 场所名称 | 文本 | - |
| 场所分类 | 文本 | - |
| 详细地址 | 文本 | - |
| 状态 | 标签 | 草稿/在用 |

##### 1.4 业务规则
- 删除仅草稿状态可操作

##### 1.5 特殊需求
- 导出Excel：是

---

#### 2. 新增/编辑弹窗

##### 2.1 页面按钮
- 保存按钮
- 取消按钮

##### 2.2 表单字段
| 字段名称 | 控件类型 | 必填 | 说明 |
|----------|----------|------|------|
| 场所名称 | 文本框 | 是 | 最大50字符 |
| 场所分类 | 下拉框 | 是 | 选项：企业/个体/政府 |
| 区划名称 | 下拉框 | 是 | 树选择 |
| 咨询电话 | 文本框 | 否 | 11位手机号 |

##### 2.3 业务规则
- 详细地址 = 区划名称 + 街道名称 + 路/弄/号（自动拼接）
- 场所名称同一类型下不能重复

##### 2.4 特殊需求
- 版本管理：修改后版本+1，历史保留
